# Mini C Compiler

## Overview

The Mini C Compiler is a browser-based application that allows users to write, compile, and analyze C programs through an interactive and user-friendly interface. The project demonstrates fundamental compiler concepts while providing a responsive coding environment with multiple output views and theme support.

---

## Features

- Interactive code editor
- Compile C programs directly in the browser
- Lexical analysis
- Syntax analysis
- Error detection and reporting
- Multiple output tabs
- Light and Dark theme support
- Responsive user interface
- Clean and intuitive design

## Installation

```bash

```

### Navigate to the Project Folder

```bash
cd Mini-C-Compiler
```

### Open the Project

Open `index.html` in your preferred web browser.

If your project requires a local server:

```bash
python -m http.server
```

Then visit:

```

```

## How to Use

1. Open the application.
2. Write or paste your C program into the editor.
3. Click the **Compile** button.
4. View the compiler output and any error messages.
5. Switch between output tabs to analyze the compilation results.
6. Toggle between Light and Dark mode if supported.
